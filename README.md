# 22mic7228

Notification platform submission project containing:

- reusable logging middleware
- notification system design
- backend API service
- frontend React application

## Structure

```text
logging_middleware/
notification_system_design.md
notification_app_be/
notification_app_fe/
```

## Setup

Install dependencies from the repository root:

```bash
npm install
```

Create environment files from the provided examples before running the apps.

## Run

Backend:

```bash
npm run dev:be
```

Frontend:

```bash
npm run dev:fe
```

The frontend is configured to run on `http://localhost:3000`.
